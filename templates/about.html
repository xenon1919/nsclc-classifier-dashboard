About the NSCLC Classifier Tool

Understand how this tool works and how it helps in identifying lung cancer types.

üî¨ What is this Tool?

The NSCLC Classifier is a web-based AI-powered tool designed to classify non-small cell lung cancer (NSCLC) types based on histopathological image inputs.

It uses a deep learning model trained on the LC25000 dataset and can distinguish between LUAD (Adenocarcinoma), LUSC (Squamous Cell Carcinoma), and Normal lung tissue.

üß† How It Works
  • You upload a lung histopathology image.
  • The model processes and classifies it into one of the three classes.
  • You receive an instant prediction with a confidence score.
